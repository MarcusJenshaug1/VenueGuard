# Frontend-rammeverk for "Utestengt"

## Valgt rammeverk: React

### Begrunnelse
- **Moderne og populært**: React er det mest brukte frontend-rammeverket globalt, med stort økosystem og god støtte.
- **Komponentbasert**: Gir god struktur og gjenbrukbarhet for komplekse grensesnitt.
- **God integrasjon med backend**: Lett å koble mot RESTful API-er fra Node.js/Express.
- **Rask utvikling**: Mange ferdige UI-biblioteker og verktøy (f.eks. Material UI, Ant Design).
- **Støtte for sikkerhet og personvern**: Enkelt å implementere tilgangsstyring og sikre brukergrensesnitt.
- **Skalerbarhet**: Egnet for både små og store prosjekter.
- **God dokumentasjon og stort utviklermiljø**: Lett å finne hjelp og eksempler.

### Alternativer vurdert
- **Angular**: Meget robust, men mer tungvint for små team og mindre prosjekter.
- **Vue**: Lettvekt og enkel, men React har større marked og flere ressurser.
