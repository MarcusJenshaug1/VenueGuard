# Arbeidsplan: Foranalyse og arkitektur

Dette dokumentet gir en detaljert arbeidsplan for punkt 1 i utviklingssjekklisten til "Utestengt".

---

## 1.1 Teknisk og juridisk kravkartlegging
- [ ] Identifisere relevante lover (GDPR, diskrimineringslov, sikkerhetskrav)
- [ ] Kartlegge datatyper, lagring, deling og sletting
- [ ] Avklare roller og ansvar (systemeier, kunde, bruker)
- [ ] Dokumentere kravene i et eget notat

## 1.2 Valg av teknologi og arkitektur
- [ ] Velge backend-teknologi (f.eks. Node.js, .NET, Python)
- [ ] Velge frontend-rammeverk (f.eks. React, Angular, Vue)
- [ ] Velge database (f.eks. PostgreSQL, MySQL, MongoDB)
- [ ] Definere hosting (skytjeneste, serverplassering i EØS)
- [ ] Skissere systemarkitektur (moduler, API, sikkerhet)

## 1.3 Definere datamodell
- [ ] Lage ER-diagram/tabellstruktur for:
    - Kunde
    - Venue
    - Person
    - Utestengelse
    - Hendelse
    - Roller/brukere
- [ ] Beskrive relasjoner og nøkkelattributter
- [ ] Dokumentere dataminimering og personvernmekanismer
