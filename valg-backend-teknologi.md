# Backend-teknologi for "Utestengt"

## Valgt teknologi: Node.js med Express

### Begrunnelse
- **Moderne, populær og godt dokumentert**: Node.js er mye brukt til webapplikasjoner og har stort økosystem.
- **Skalerbarhet**: God støtte for asynkrone operasjoner og høy samtidighet.
- **Sikkerhet**: Mange tilgjengelige moduler for autentisering, kryptering og logging.
- **API-støtte**: Express gir fleksibel og rask utvikling av RESTful API-er.
- **Utviklingshastighet**: Rask prototyping og mange ferdige pakker.
- **Støtte for dataminimering og personvern**: Enkelt å implementere sikkerhetsmekanismer og personvernrutiner.
- **God integrasjon med frontend-rammeverk**: Lett å koble mot React, Angular eller Vue.

### Alternativer vurdert
- **.NET (C#)**: Meget robust og sikkert, men tyngre og mer ressurskrevende for små team.
- **Python (Django/Flask)**: Rask utvikling, men Node.js gir bedre samtidighet og API-støtte for denne typen løsning.
